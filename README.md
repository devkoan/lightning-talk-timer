# lightning-talk-timer

Free countdown timer for lightning talks and developer presentations.

Live at: **https://devkoan.github.io/lightning-talk-timer/**

## Features

- 5 / 7 / 10 / 15 min presets + custom duration
- Ring progress animation
- 1 min warning (orange) + 30 sec warning (red flash) + audio beeps
- Focus mode — hides everything except the timer
- Slide pacing guide — highlights current slide in real time
- Lightning talk tips

## Deploy

Single `index.html` — no build step, no dependencies.

1. Clone this repo
2. Settings → Pages → Source: `main` branch, `/ (root)`
3. Live at `https://devkoan.github.io/lightning-talk-timer/`
